一、相关知识点

1. 根式不等式

√f(x) < g(x) ⇔ f(x) ≥ 0，g(x) > 0，并且 f(x) < [g(x)]²
√f(x) > g(x) ⇔ ① f(x) ≥ 0，g(x) < 0；或 ② f(x) ≥ 0，g(x) ≥ 0，且 f(x) > [g(x)]²
平方前必须先判断两边的符号，不能直接平方。

2. 分式不等式

先移到同一边并通分，再用“零点、无定义点、符号表”求解。分母为零的点永远不能取。

3. 含绝对值函数的最值

形如 Σaᵢ|x−xᵢ|（aᵢ>0）的函数，在其“加权中位数”处取得最小值。

二、图片中的例题

例题3

解不等式：(x−3)(x−8)(x+2)(x+4)² ≥ 0。

(x+4)²≥0，并且只在 x=−4 时等于0。除 x=−4 外，原式符号由

(x−3)(x−8)(x+2)

决定。三个一次因式的零点依次为 −2、3、8，作符号表：

区间(−∞,−2)(−2,3)(3,8)(8,+∞)
符号−+−+

解集：{−4} ∪ [−2,3] ∪ [8,+∞)。

例题7

已知 a>b，求不等式 1x−a + 1x−b ≥1 的解所构成区间的长度之和。

移项并通分：

1x−a + 1x−b−1 = −x²+(a+b+2)x−(ab+a+b)(x−a)(x−b)

设分子两根为 r₁<r₂，则

r₁,₂ = [a+b+2 ∓ √((a−b)²+4)]/2

可验证其顺序为 b<r₁<a<r₂。符号分析得解集：

(b,r₁] ∪ (a,r₂]

区间长度之和：

(r₁−b)+(r₂−a)=(r₁+r₂)−(a+b)=2

答案：2。

三、课后练习第1题

（1）解不等式 |x²−5x+6| < x²−4

|x²−5x+6| < x²−4。

因为绝对值非负，所以右边必须为正：

x²−4>0 ⇔ x<−2 或 x>2

利用 |A|<B ⇔ −B<A<B（其中 B>0）：

−(x²−4)<x²−5x+6<x²−4

右侧不等式：

x²−5x+6<x²−4 ⇔ −5x<−10 ⇔ x>2

左侧不等式：

x²−5x+6>−x²+4 ⇔ 2x²−5x+2>0
(2x−1)(x−2)>0 ⇔ x<1/2 或 x>2

综合所有条件，得到：

解集：(2,+∞)。

（2）解不等式 (x²−x−2)(√(2x+1)+1) ≤ 0

(x²−x−2)(√(2x+1)+1) ≤ 0。

定义域：

2x+1≥0 ⇔ x≥−1/2

在定义域内，√(2x+1)+1>0，因此只需：

x²−x−2≤0 ⇔ (x−2)(x+1)≤0 ⇔ −1≤x≤2

与定义域取交集：

解集：[−1/2,2]。

（3）解分式不等式

1x+4 + 1x+5 > 1x+6 + 1x+3。

两边分别配对：

[1/(x+4)−1/(x+3)] + [1/(x+5)−1/(x+6)] > 0
−1/[(x+4)(x+3)] + 1/[(x+5)(x+6)] >0

通分后分子为：

(x+3)(x+4)−(x+5)(x+6)=−4x−18=−2(2x+9)

所以：

−2(2x+9)(x+3)(x+4)(x+5)(x+6)>0

临界点依次为 −6、−5、−9/2、−4、−3。作符号表可得：

解集：(−∞,−6) ∪ (−5,−9/2) ∪ (−4,−3)。

四、第2题：解集为 R 的参数范围

不等式 (m²−2m−3)x²−(m−3)x−1<0 的解集为 R，求实数 m 的范围。

记二次项系数：

A=m²−2m−3=(m−3)(m+1)

当 A<0 时，二次函数开口向下。要使其对所有实数恒小于0，还需判别式 Δ≤0：

−1<m<3
Δ=[−(m−3)]²−4(m²−2m−3)(−1)
Δ=(m−3)²+4(m−3)(m+1)=(m−3)(5m+1)

在 −1<m<3 内，m−3<0，因此 Δ≤0 等价于：

5m+1≥0 ⇔ m≥−1/5

另需检查退化情形：m=3 时，原不等式变为 −1<0，对所有 x 成立；m=−1 时变为一次不等式，不能对所有 x 成立。

m∈[−1/5,3]。

五、第3题：解集关系

已知不等式 x²−ax+2<0 的解集为 A，求满足下列条件的 a：
（1）A=(1,2)；（2）(1,2)⊆A；（3）A⊆(1,2)。

若方程 x²−ax+2=0 的两根为 r₁<r₂，则：

A=(r₁,r₂)，r₁+r₂=a，r₁r₂=2

（1）A=(1,2)

两根就是1和2，因此 a=1+2=3。

a=3。

（2）(1,2)⊆A

要使整个开区间 (1,2) 都落在负值区间内，只需端点处函数值不大于0：

f(1)=3−a≤0，f(2)=6−2a≤0

两者均给出 a≥3。

a≥3。

（3）A⊆(1,2)

要求两根满足 1≤r₁<r₂≤2，并且 r₁r₂=2。

令 r₁=t，则 r₂=2/t，其中 1≤t<√2。于是：

a=t+2/t

该式在区间 [1,√2) 上从3递减趋近2√2，但不能取2√2（否则两根重合，解集为空）。

2√2<a≤3。

六、第4题：含参数二次不等式

解关于 x 的不等式：kx²+kx+1≤0（k∈R）。

判别式：

Δ=k²−4k=k(k−4)

① k=0

1≤0

无解。

② k>0

抛物线开口向上，必须 Δ≥0，即 k≥4。

x₁=[−k−√(k²−4k)]/(2k)，x₂=[−k+√(k²−4k)]/(2k)

当 k≥4 时，x∈[x₁,x₂]；当 0<k<4 时无解。

③ k<0

此时 Δ>0，抛物线开口向下，“≤0”在两根外侧成立。由于分母2k<0，较小的根为：

x₁=[−k+√(k²−4k)]/(2k)，x₂=[−k−√(k²−4k)]/(2k)

x∈(−∞,x₁]∪[x₂,+∞)。

七、第5题：含参数分式不等式

解关于 x 的不等式： a(x−1)/(x−2)>1（a≠1）。

移项通分：

a(x−1)x−2−1 = (a−1)x+(2−a)x−2>0

分子零点：

x₀=(a−2)/(a−1)

并比较 x₀ 与2：

x₀−2=−a/(a−1)
a 的范围x₀与2的顺序解集
a<0x₀<2(x₀,2)
a=0—∅
0<a<12<x₀(2,x₀)
a>1x₀<2(−∞,x₀)∪(2,+∞)

其中 x=2 始终无定义，x=x₀ 使分子为0，也不能取。

八、第6题：绝对值和的最小值

（2011北京）求 f(x)=|x−1|+|2x−1|+⋯+|2011x−1| 的最小值。

将每一项改写：

|ix−1|=i|x−1/i|

所以：

f(x)=Σ(i=1到2011) i|x−1/i|

这是加权距离和。点 1/i 的权重为 i，最小值在加权中位数处取得。

总权重：

1+2+⋯+2011=2011×2012/2=2,023,066

总权重的一半为1,011,533。寻找整数 k，使点 x=1/k 两侧的权重都不超过这一半：

1+2+⋯+1421=1,010,331

精确计算：

1+2+⋯+1421=1,010,331；1423+⋯+2011=1,011,313

两侧权重均不超过1,011,533，因此加权中位数为：

x=1/1422

此时：

f(1/1422)=Σ|i−1422|/1422
=[1+2+⋯+1421 + 1+2+⋯+589]/1422
=[1421×1422/2 + 589×590/2]/1422
=592043/711

最小值为 592043/711，约等于 832.691。

九、例题11：加权绝对值函数的最值规律

探索 f(x)=a₁|x−x₁|+a₂|x−x₂|+⋯+aₙ|x−xₙ|（aᵢ>0）的最小值。

把 x 看成数轴上的活动点，每一项 aᵢ|x−xᵢ| 表示“距离 × 权重”。

结论：当 x 位于这些点的加权中位数处时，函数取得最小值。

加权中位数的判断方法：该点左侧所有权重之和不超过总权重的一半，右侧所有权重之和也不超过总权重的一半。

原因是：x 向右移动一个很小距离时，左边各点对应的距离增加，右边各点对应的距离减少。只有当左右两侧的总权重达到平衡时，继续向任一方向移动都不会使总和下降。
