二、学习主线：这一章到底在学什么？

主线1： 先认识任意角和弧度制，知道角不再只局限于 0° 到 180°。
主线2： 再借助坐标系和单位圆，理解 sin、cos、tan 为什么可以表示任意角。
主线3： 最后学习诱导公式，把“复杂角”化成“熟悉的锐角”，快速求值。
一句话概括： 本章的核心就是：把大角、负角、特殊形式的角，转化成熟悉的小锐角来计算。

三、弧度制与任意角的三角比

1. 任意角：角可以比 360° 更大，也可以是负的

在平面直角坐标系中，通常把正 x 轴作为角的 始边。从始边转到终边，得到一个角。

  • 逆时针旋转：得到正角。
  • 顺时针旋转：得到负角。
  • 转一整圈后再继续转：可以得到大于 360° 的角。
图1：任意角示意图——同一个始边，可以转出正角、负角和超过一周的角。

终边相同角

如果两个角转完后，终边落在同一条射线 上，那么它们叫作终边相同的角。

β = α + k·360°（k∈Z）
或 β = α + 2kπ（k∈Z）

例如：

  • 30°、390°、−330° 终边相同；
  • π/6、13π/6、−11π/6 终边相同。
理解重点： 三角函数值只与终边位置有关，所以终边相同的角，三角函数值也相同。

2. 弧度制：为什么 180° 等于 π？

弧度制不是“另一套难记的单位”，而是由圆本身自然产生的角度单位。

图2：弧度制的本质——角的大小 = 弧长 ÷ 半径。

推导：180° = π

整圆一周的弧长是 2πr，所以整圆对应的弧度数是：

2πr / r = 2π

也就是说：

360° = 2π

两边同时除以 2：

180° = π

常用互化公式

1° = π/180（弧度）
1（弧度） = 180°/π

所以：

  • 角度 → 弧度：乘 π/180
  • 弧度 → 角度：乘 180/π

3. 任意角的三角比：为什么单位圆最重要？

设角 α 的终边上一点 P(x, y)，点 P 到原点的距离为 r，则：

sinα = y/r，cosα = x/r，tanα = y/x（x ≠ 0）

如果选取 单位圆（即 r = 1），就会变得非常简单：

P(cosα, sinα)
图3：在单位圆上，横坐标就是 cosα，纵坐标就是 sinα。

最关键的一句话

单位圆上点的横坐标 = cosα，纵坐标 = sinα。

所以很多诱导公式，其实就是看图形对称以后，坐标怎么变。

为什么 tanα = y/x？

因为：

tanα = sinα / cosα = (y/r) / (x/r) = y/x

因此，当 x = 0 时，tanα 没有意义。

4. 各象限符号：先看象限，再定正负

图4：四象限符号图。

记忆口诀

一全正，二正弦，三正切，四余弦。

  • 第一象限：三个都正；
  • 第二象限：只有 sin 正；
  • 第三象限：只有 tan 正；
  • 第四象限：只有 cos 正。

最常见失分点

  1. 会化角，但最后符号写反。
  2. 把 330° 看成 cos 为负，实际上它在第四象限，cos 为正。
  3. 忘记 tan 的符号是由 sin 和 cos 共同决定的。

5. 基础例题

例1：把下列角度化成弧度：60°，225°，−150°

查看解析
角度化弧度：统一乘 π/180
  • 60° = 60×π/180 = π/3
  • 225° = 225×π/180 = 5π/4
  • −150° = −150×π/180 = −5π/6

例2：求 tan225°

查看解析
先把角写成熟悉形式：225° = 180° + 45°

第三象限中 tan 为正，所以：

tan225° = tan(180°+45°) = tan45° = 1

例3：求 sin(−30°)、cos330°

查看解析

（1）负角先看奇偶性：

sin(−30°) = −sin30° = −1/2

（2）330° = 360° − 30°，在第四象限，cos 为正：

cos330° = cos(360°−30°) = cos30° = √3/2

四、诱导公式

什么是诱导公式奇变偶不变公式总表图形推导典型例题

1. 什么是诱导公式？

诱导公式就是把任意角的三角函数，转化为锐角的三角函数。

比如：

sin150° = sin(180°−30°) = sin30° = 1/2

原来不会直接求的 150°，被转化成会求的 30°。

2. 最重要的口诀：奇变偶不变，符号看象限

“奇变偶不变”是什么意思？

当角写成 k×90° ± α 的形式时：

  • k 是偶数（0、2、4……）：函数名称不变；
  • k 是奇数（1、3、5……）：函数名称改变。

例如：

  • sin(180°−α)：180° = 2×90°，2 是偶数，所以 sin 不变；
  • sin(90°+α)：90° = 1×90°，1 是奇数，所以 sin 变 cos。

“符号看象限”是什么意思？

先确定角在哪个象限，再决定最终符号。

例如：

  • cos(180°−α)：第二象限，cos 为负，所以结果是 −cosα；
  • sin(90°+α)：第二象限，sin 为正，所以结果是 +cosα。

4. 用图形看懂诱导公式

图6：π−α 与 α 关于 y 轴对称，所以 sin 不变，cos 变号。

由图得到

sin(π−α)=sinα
cos(π−α)=−cosα
tan(π−α)=−tanα

为什么？

  • 纵坐标没变，所以 sin 不变；
  • 横坐标变成相反数，所以 cos 变号；
  • tan = sin/cos，因此也变号。
图7：π+α 与 α 关于原点中心对称，所以 sin、cos 都变号。

由图得到

sin(π+α)=−sinα
cos(π+α)=−cosα
tan(π+α)=tanα

为什么 tan 不变？

tan(π+α) = (−sinα)/(−cosα) = tanα
图8：π/2−α 是 α 的余角，因此 sin 与 cos 交换。

由图得到

sin(π/2−α)=cosα
cos(π/2−α)=sinα

这就是为什么涉及 90° 时，经常会“变名”。

5. 典型例题（按步骤解释更清晰）

例4：求 sin150°，cos150°，tan150°

查看分步解析
第1步： 150° = 180° − 30°
第2步： 180° 是偶数个 90°，所以函数名称不变
第3步： 150° 在第二象限，sin 正，cos 负，tan 负
sin150° = sin30° = 1/2
cos150° = −cos30° = −√3/2
tan150° = −tan30° = −√3/3

例5：求 sin210°，cos210°，tan210°

查看分步解析
第1步： 210° = 180° + 30°
第2步： 180° 是偶数个 90°，函数名称不变
第3步： 210° 在第三象限，sin 负，cos 负，tan 正
sin210° = −sin30° = −1/2
cos210° = −cos30° = −√3/2
tan210° = tan30° = √3/3

例6：化简 sin(90°+α)

查看分步解析
第1步： 90° 是 1 个 90°，属于奇数，所以要“变名”
第2步： sin 变成 cos
第3步： 若 α 为锐角，则 90°+α 在第二象限，sin 为正
sin(90°+α)=cosα

例7：化简 cos(270°−α)

查看分步解析
第1步： 270° = 3×90°，3 是奇数，所以要“变名”
第2步： cos 变成 sin
第3步： 270°−α 在第三象限，cos 为负
cos(270°−α)=−sinα

五、练习题

建议先独立完成，再展开答案对照。答案也保留了较清楚的说明。

A组：基础练习

  1. 将下列角度化成弧度：120°，300°，−225°。
  2. 将下列弧度化成角度：π/3，5π/3，−7π/6。
  3. 写出与 45° 终边相同的所有角。
  4. 求 sin135°。
  5. 求 cos240°。
  6. 求 tan315°。
查看 A组答案
  1. 120° = 2π/3，300° = 5π/3，−225° = −5π/4。
  2. π/3 = 60°，5π/3 = 300°，−7π/6 = −210°。
  3. 45° + k·360°（k∈Z）或 π/4 + 2kπ。
  4. 135° = 180° − 45°，第二象限 sin 为正，所以 sin135° = √2/2。
  5. 240° = 180° + 60°，第三象限 cos 为负，所以 cos240° = −1/2。
  6. 315° = 360° − 45°，第四象限 tan 为负，所以 tan315° = −1。

B组：提高练习

  1. 求 sin225°，cos225°，tan225°。
  2. 求 sin(−150°)，cos(−150°)。
  3. 化简：sin(180°+α)，cos(180°−α)，tan(360°−α)。
  4. 化简：sin(90°−α)+cos(90°−α)。
  5. 已知 sinα=3/5，且 α 在第二象限，求 cosα 与 tanα。
  6. 已知 cosα=−4/5，且 α 在第三象限，求 sinα 与 tanα。
查看 B组答案
  1. 225° = 180° + 45°，第三象限：sin225° = −√2/2，cos225° = −√2/2，tan225° = 1。
  2. sin(−150°) = −sin150° = −1/2；cos(−150°) = cos150° = −√3/2。
  3. sin(180°+α)=−sinα；cos(180°−α)=−cosα；tan(360°−α)=−tanα。
  4. sin(90°−α)+cos(90°−α)=cosα+sinα。
  5. cos²α = 1−sin²α = 1−9/25 = 16/25，所以 |cosα| = 4/5。第二象限 cos<0，因此 cosα = −4/5；tanα = (3/5)/(−4/5)=−3/4。
  6. sin²α = 1−cos²α = 1−16/25 = 9/25，所以 |sinα|=3/5。第三象限 sin<0，因此 sinα = −3/5；tanα = (−3/5)/(−4/5)=3/4。

C组：综合练习

  1. 证明：sin(π+α)=−sinα。
  2. 证明：cos(π−α)=−cosα。
  3. 若 tanα=2，求 tan(π+α) 与 tan(2π−α)。
  4. 求值：sin750°、cos(−420°)、tan1110°。
  5. 化简：sin(270°+α)+cos(180°−α)。
  6. 若 θ 与 −30° 终边相同，求 sinθ、cosθ、tanθ。
查看 C组答案
  1. 由图7，角 π+α 与 α 关于原点中心对称，所以纵坐标变为相反数，故 sin(π+α)=−sinα。
  2. 由图6，角 π−α 与 α 关于 y 轴对称，所以横坐标变为相反数，故 cos(π−α)=−cosα。
  3. tan(π+α)=tanα=2；tan(2π−α)=−tanα=−2。
  4. sin750° = sin(720°+30°)=1/2；cos(−420°)=cos(−60°)=1/2；tan1110°=tan(1080°+30°)=√3/3。
  5. sin(270°+α)+cos(180°−α)=−cosα−cosα = −2cosα。
  6. 终边相同的角三角函数值相同，所以 sinθ = −1/2，cosθ = √3/2，tanθ = −√3/3。

六、最后总结：做题时最稳妥的方法

步骤1

把角改写成
0°、90°、180°、270°、360° ± α 的形式。

步骤2

看这个基准角是几个 90°，判断是
奇变 还是 偶不变。

步骤3

看原角所在象限，决定最终符号。

步骤4

把结果化成熟悉的锐角函数值，如 30°、45°、60°。

重点记忆

一全正，二正弦，三正切，四余弦。
